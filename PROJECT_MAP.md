# 🟢 مشروع "فرص خضراء" - المساعد الذكي

## 1. الهدف

بناء مساعد ذكي (Chatbot) يجيب على أسئلة المستخدمين عن الفرص المتاحة (منح، وظائف، تدريب) على منصة "فرص خضراء"، باستخدام Python + Flask + Google Gemini API (الطبقة المجانية).

---

## 2. الافتراضات الأساسية

- المشروع يعمل **محلياً (Local)** على جهاز المطور
- **Google Gemini API** هو النموذج اللغوي المختار (طبقة مجانية، لا حاجة لبطاقة ائتمان)
- البيانات: ملف JSON واحد يحتوي على 15–20 فرصة (منح، وظائف، تدريب)
- الجمهور: متخصص أمن سيبراني، خبرة محدودة في AI
- لغة الواجهة: **العربية** (البرومبتات والردود)
- لغة الكود: **الإنجليزية** (أسماء المتغيرات، الدوال، التعليقات إن وجدت)
- المدة: **3 أيام**
- لا تكاليف APIs إطلاقاً

## 5. تدفق المستخدم (User Journey)

```
[المستخدم] → يكتب سؤالاً بالعربية
         → [Flask Route - /chat]
         → [Rate Limiter - 10 طلبات/دقيقة]
         → [Input Validator - تنظيف وفحص]
         → [Data Service - استخراج المعلومات ذات الصلة من JSON]
         → [LLM Service - بناء الـ Prompt + إرسال إلى Gemini]
         → [Gemini API - رد بالعربية]
         → [Logging - تسجيل الطلب والرد]
         → [المستخدم] ← الرد يظهر في الواجهة
```

## 6. خطة العمل (Milestones)

### اليوم الأول: البنية التحتية + البيانات

- [ ] إنشاء `config.py` يحتوي على الإعدادات الأساسية
- [ ] إنشاء `data/opportunities.json` بـ 15–20 فرصة (منح، وظائف، تدريب)

### اليوم الثاني: المنطق الأساسي + AI

**الهدف القابل للتحقق:** المساعد يرد على أسئلة الفرص باستخدام Gemini

- [ ] إنشاء `services/data_service.py` (قراءة JSON، البحث عن الفرص)
- [ ] إنشاء `services/llm_service.py` (الاتصال بـ Gemini API)
- [ ] الحصول على مفتاح API من Google AI Studio (مجاني)
- [ ] إنشاء ملف `.env` وتخزين المفتاح
- [ ] بناء نظام الـ Prompt بالعربية مع سياق الفرص
- [ ] إضافة route `/chat` (POST) لاستقبال الأسئلة
- [ ] ربط الذكاء الاصطناعي مع واجهة المستخدم

## 10. ملاحظات أمنية (مهمة)

- **لا تشارك ملف `.env`** أبداً (فيه API key)
- `API key` مجاني ويمكن إعادة توليده من AI Studio
- الحد الأقصى لـ Gemini Free Tier: 10 طلبات/دقيقة، 250 طلب/يوم
- Rate Limiter يمنع استغلال الـ API key
- Input Validator يمنع إدخال أكواد ضارة
- هذا تطبيق تجريبي (prototype)، ليس جاهزاً للإنتاج
